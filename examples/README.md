# Examples

Thư mục này lưu các ví dụ theo từng môn học để tham khảo nhanh khi làm bài mới.

## Cách tổ chức

```text
examples/
└── <ten-mon-hoc>/
    └── <ma-de-hoac-chu-de>/
        ├── README.md
        ├── bao-cao-mau.md
        └── ghi-chu-review.md
```

## Nguyên tắc dùng example

- Không copy nguyên văn sang bài mới.
- Dùng để tham khảo cấu trúc, cách đặt vấn đề, cách viết chương, cách dùng bảng/hình.
- Luôn thay đổi bối cảnh, liên hệ thực tế và tài liệu tham khảo theo đề bài mới.
- Trước khi nộp, rà soát lại bằng `templates/review_checklist.md`.
